	•	1️⃣ What is the difference between var, let, and const?
	Ans: Var: It can create unexpected bugs because it ignores block scope.
	•	     Let: Modern way to declare a variable, Block scoped (works inside { }, Can be updated but cannot be redeclared.
         const: Used for a constant value, block-scoped, cannot be updated or redeclared.
	•	2️⃣ What is the spread operator (...)?
  Ans: The spread operator (...) in JavaScript is used to expand or copy elements from an array or object.
	•	3️⃣ What is the difference between map(), filter(), and forEach()?
  Ans: 
  map()
1. Used to change or transform every element in an array.
2. Returns a new array with the transformed values.
   
   filter()
   1. Used to select elements based on a condition
   2. Returns a new array.
  
   forEach()
   1. Used to run a function for each element
   2. Does NOT return a new array.
